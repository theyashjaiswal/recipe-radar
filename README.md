## Instructions to Run

1. **Clone the repository:**

   ```bash
   git clone <repository-link>
   cd recipe-radar

   ```

2. **Install dependencies:**

   ```bash
   npm install

   ```

3. **Run the development server:**

   ```bash
   npm start

   ```

4. **View the application in your browser:**
   Open http://localhost:3000 to view the app locally.
